# PersonLog — 单人工作日志

一个 Windows 桌面工具，双击即用，不安装、不联网、不写注册表。按天记录工作日志，支持模糊查询历史。

## 功能

- **日期选择**：顶部日期控件，切换日期自动加载当天日志
- **日志编辑**：富文本编辑区，RTF 格式存储，支持中文排版
- **按天存储**：每条日志独立文件，目录结构 `LOG/YYYY/MM/DD.DIL`
- **模糊查询**：输入关键词 → 点「查询」→ 左侧列出匹配日期 → 单击跳转
- **Ctrl+F** 聚焦搜索框，**Ctrl+S** 保存

## 使用

1. 下载 [LogEditor.exe](../../releases) 放到你喜欢的目录
2. 双击运行，自动在同目录创建 `LOG` 文件夹
3. 选择日期 → 写日志 → 点「保存」
4. 需要查历史：输入关键词 → 点「查询」→ 单击结果跳转

> 如果你之前有「工作日志单机版 1.9」的记录，把它的 `LOG` 文件夹复制到 exe 同级目录即可无缝兼容。

## 编译

需要 MinGW（或 MSVC）。

```bat
:: 一行编译
build.bat
```

或手动：

```bat
:: 1. 生成图标资源
windres icon.rc icon.o

:: 2. 编译
g++ -O2 -std=c++11 -mwindows -static LogEditor.cpp icon.o -o LogEditor.exe -lcomctl32 -lshell32
```

纯 C++ / Win32 API，零外部依赖，静态链接，编译产物约 1.5 MB。

## 日志格式

日志文件为 RTF (Rich Text Format)，GBK 编码，存储在：

```
LOG/
├── 2025/
│   ├── 01/
│   │   ├── 01.DIL
│   │   ├── 02.DIL
│   │   └── ...
│   └── 02/
└── 2026/
```
